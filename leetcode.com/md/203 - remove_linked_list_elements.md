# 203. Remove Linked List Elements

Remove all elements from a linked list of integers that have value _**val**_.

**Example**

_**Given**_: 1 --> 2 --> 6 --> 3 --> 4 --> 5 --> 6, _**val**_ = 6

_**Return**_: 1 --> 2 --> 3 --> 4 --> 5
